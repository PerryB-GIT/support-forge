Calendly Integration Options

Schedule Your Consultation

Choose from multiple integration methods to provide the best booking experience for your clients.

RECOMMENDED

Option 1: Inline Widget

Embed Calendly directly into your page for a seamless booking experience. No redirects needed.

Option 2: Popup Button

Open Calendly in a popup overlay when users click a button.

<button onclick="Calendly.initPopupWidget({url: 'https://calendly.com/perry-bailes'});return false;">Book a Meeting</button>

Option 3: Direct Link

Simple link that opens Calendly in a new tab or current page.

Open Calendly <a href="https://calendly.com/perry-bailes" target="_blank">Schedule</a>

Option 4: Custom API Integration

Use the Calendly API for complete control over the booking experience.

GET /api/calendly/event-types GET /api/calendly/availability POST /api/calendly/book

Note: Requires backend server (Express.js setup included in this project).

Implementation Guide

Step 1: Add Calendly JavaScript to your HTML (before closing </body> tag):

<script type="text/javascript" src="https://assets.calendly.com/assets/external/widget.js" async></script>

Step 2: Add Calendly CSS to your <head> section:

<link href="https://assets.calendly.com/assets/external/widget.css" rel="stylesheet">

Step 3: Customize the appearance using URL parameters:

?hide_gdpr_banner=1&primary_color=dc2626&text_color=f9fafb&background_color=141419

Step 4: Replace the custom form in index.html with your chosen integration method.

Why Use Calendly Integration?

  • Real-time Availability: Automatically syncs with your calendar to show only available slots
  • Automatic Reminders: Sends email and SMS reminders to reduce no-shows
  • Time Zone Detection: Automatically detects and converts to user's local time zone
  • Calendar Integration: Syncs with Google Calendar, Outlook, iCloud, and more
  • Professional Experience: Branded, mobile-responsive booking flow
  • No Double Bookings: Prevents scheduling conflicts automatically
