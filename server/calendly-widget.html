Calendly Integration Options

Option 1: Inline Widget (Recommended)

Embed Calendly directly in your page:

Code:

<!-- Add to your HTML where you want the widget -->
<div class="calendly-inline-widget" 
     data-url="https://calendly.com/perry-bailes?hide_gdpr_banner=1&primary_color=c97c4b"
     style="min-width:320px;height:700px;">
</div>
<script src="https://assets.calendly.com/assets/external/widget.js" async></script>

Option 2: Popup Widget

Open Calendly in a popup when user clicks a button:

Code:

<!-- Add Calendly CSS -->
<link href="https://assets.calendly.com/assets/external/widget.css" rel="stylesheet">

<!-- Add Calendly JS -->
<script src="https://assets.calendly.com/assets/external/widget.js" async></script>

<!-- Button to trigger popup -->
<button onclick="Calendly.initPopupWidget({url: 'https://calendly.com/perry-bailes'});">
  Schedule a Consultation
</button>

Option 3: Badge Widget

Floating badge in corner of page:

Code:

<!-- Add to bottom of body -->
<script src="https://assets.calendly.com/assets/external/widget.js" async></script>
<script>
window.onload = function() {
  Calendly.initBadgeWidget({
    url: 'https://calendly.com/perry-bailes',
    text: 'Schedule a Call',
    color: '#c97c4b',
    textColor: '#ffffff'
  });
}
</script>

Option 4: Prefilled Data

Pre-fill visitor information from your form:

Code:

<script>
// After collecting user info from your form
function openCalendlyWithPrefill(name, email) {
  Calendly.initPopupWidget({
    url: 'https://calendly.com/perry-bailes',
    prefill: {
      name: name,
      email: email,
      customAnswers: {
        a1: 'Answer to question 1'  // If you have custom questions
      }
    }
  });
}
</script>

Customization Options

URL parameters you can use:

  • primary_color=c97c4b - Match your copper theme
  • hide_gdpr_banner=1 - Hide cookie banner
  • hide_landing_page_details=1 - Minimal view
  • hide_event_type_details=1 - Hide event description
  • background_color=0a0a10 - Dark background
  • text_color=f9fafb - Light text
