## Video Types & Production Methods

### Type 1: Talking Head (Avatar Presenter)
**Use for:** Introductions, explanations, transitions
**Tool:** HeyGen via Canva
**Process:**
```
Script (.md) → HeyGen → Avatar video → Canva assembly
```

### Type 2: Screen Walkthrough
**Use for:** Software demos, tutorials, configurations
**Tool:** Guidde or manual capture + AI voiceover
**Process:**
```
Screen record → Guidde auto-edit → Add AI voiceover → Export
```

### Type 3: Animated Explainer
**Use for:** Concepts, diagrams, architecture
**Tool:** Canva + Veo 3 for B-roll
**Process:**
```
Create slides → Add animations → Veo 3 B-roll → AI voiceover
```

### Type 4: Hybrid (Avatar + Screen)
**Use for:** Most tutorial content
**Tool:** Combination
**Process:**
```
1. Avatar intro (HeyGen)
2. Screen walkthrough (Guidde)
3. Avatar summary (HeyGen)
4. Assemble in Canva
```

## Automation Opportunities

### 1. Script → Avatar Video Pipeline

```python
# Automated HeyGen video generation
def generate_avatar_video(script_path: str) -> str:
    """Generate avatar video from script markdown."""

    # Read script
    with open(script_path) as f:
        script = f.read()

    # Extract narration sections
    narration = extract_narration(script)

    # Call HeyGen API
    video_url = heygen.create_video(
        avatar_id="sparky-v1",
        script=narration,
        voice_id="professional-friendly",
        background="support-forge-studio"
    )

    return video_url
```

### 2. Batch Screen Capture Processing

```python
# Guidde API integration
def process_walkthrough(recording_path: str) -> str:
    """Process screen recording into tutorial."""

    # Upload to Guidde
    guide = guidde.create_from_recording(
        file=recording_path,
        voice="ai-professional",
        language="en",
        auto_zoom=True,
        add_highlights=True
    )

    return guide.export_video()
```

### 3. Video Assembly Automation

```python
# Canva API for assembly
def assemble_lesson(components: list) -> str:
    """Assemble video components into final lesson."""

    project = canva.create_video_project(
        template="support-forge-lesson"
    )

    for component in components:
        if component.type == "avatar":
            project.add_clip(component.url, duration=component.duration)
        elif component.type == "screen":
            project.add_clip(component.url, with_pip_avatar=True)
        elif component.type == "slide":
            project.add_slide(component.url, duration=5)

    # Add outro
    project.add_template("lesson-outro")

    return project.export()
```
