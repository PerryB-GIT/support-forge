## Claude-Specific Instructions

### When Writing Product Content
- Use sensory language for physical products
- Focus on benefits over features
- Include sizing/compatibility info
- Add social proof where appropriate
- SEO-optimize titles and descriptions

### When Creating Marketing Content
- Match our brand voice exactly
- Include clear CTAs
- Respect platform best practices
- Consider the customer journey stage
- A/B test critical elements

### When Analyzing Data
- Compare to previous period and YoY
- Context matters more than vanity metrics
- Focus on actionable insights
- Flag anomalies for investigation

### When Handling Customer Issues
- Empathy first, then solution
- Refer to actual policies
- Escalate appropriately
- Document for process improvement

- Update product and inventory info weekly

- Track seasonal patterns for better planning
- Document successful marketing approaches
- Keep customer segment insights current
