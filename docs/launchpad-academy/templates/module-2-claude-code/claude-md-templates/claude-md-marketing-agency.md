## Common Tasks

### Campaign Setup
When creating campaigns, always include:
- Clear objective aligned to client goal
- Target audience definition
- Budget allocation across channels
- Timeline with milestones
- Success metrics and benchmarks
- A/B testing plan

### Client Reporting
Standard report sections:
1. Executive summary (3-bullet highlights)
2. KPI dashboard (vs. goal, vs. last period)
3. Channel performance breakdown
4. Key insights and recommendations
5. Next month action items
6. Budget utilization

### Content Calendar
When creating content plans:
- Map to buyer journey stages
- Include platform-specific variations
- Note required assets (images, video, copy)
- Add relevant hashtags and keywords
- Include posting times based on audience data

## Workflow Shortcuts

### Requesting a Campaign Brief
"Create a campaign brief for [Client] with:
- Objective: [Awareness/Consideration/Conversion]
- Budget: $[X]
- Timeline: [Duration]
- Target audience based on our existing personas
- Recommended channel mix
- Key messages aligned to their brand"

### Creating a Report
"Generate a monthly performance report for [Client] covering:
- The date range [Month]
- Compare to previous month and same month last year
- Include insights on what's working and what to improve
- Add 3 recommended actions for next month"

### Competitor Analysis
"Analyze [Competitor Name] for [Client]:
- Their ad presence (use Meta Ad Library, Google Ads Transparency)
- Social media strategy
- SEO positioning for our target keywords
- Identify gaps we can exploit"

### Writing Ad Copy
"Write [Number] variations of [Ad Type] for [Client]:
- Platform: [Google/Meta/LinkedIn]

- Character limits: [Specify]

## Claude Specific Instructions

When I ask about:
- **"the client"** without specifying: Ask which client
- **Performance data**: Note you need actual data access or ask me to provide it
- **Ad copy**: Generate multiple variations and note character counts
- **Strategy**: Consider budget constraints and client history
- **Reports**: Use our standard template structure

Always:
- Include specific numbers and dates when possible
- Flag if something needs client approval
- Consider the client's industry regulations
- Reference brand guidelines for tone
